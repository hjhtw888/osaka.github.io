  • 護照 (效期需6個月以上)
  • Visit Japan Web (VJW) 截圖/QR Code
  • 旅遊不便險/平安險英文證明
  • 日網卡/eSIM 或 Wi-Fi 機
  • 行動電源 (多備一顆，狂拍照必備)
  • 轉接頭 (台灣電器通常可直接用，但注意電壓)

  • 短袖衣物、透氣長褲/短褲
  • 薄外套 (防冷氣、防曬)
  • 好穿耐走的運動鞋 (非常重要！)
  • 摺疊雨傘/輕便雨衣 (6月易下雨)
  • 兒童常備藥 (腸胃藥、退燒藥、OK繃)
  • 摺疊購物袋 (日本購物袋多需付費)
